# react

Add an emoji reaction to the message.

## Parameters

- `emoji` — the emoji name (e.g. `eyes`, `thumbsup`, `fire`, `100`, `wave`, `thinking_face`)

## When to use

- Someone shares something and doesn't need a reply — just acknowledgment
- Agreement with what was said
- Humor or vibes
- Signaling you've seen it without interrupting the flow

## When NOT to use

- When the person is asking a question (they need a reply, not a react)
- When there's ambiguity about what the reaction means

## Choosing the right emoji

Match the energy:
- Good news, wins → `fire`, `100`, `tada`
- Acknowledgment, "seen it" → `eyes`, `thumbsup`
- Funny → `joy`, `skull`
- Thinking about it → `thinking_face`
- Solidarity / support → `heart`, `raised_hands`

One reaction. Don't stack multiple.
